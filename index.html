:disabled

La pseudo-clase :disabled de CSS representa a cualquier elemento deshabilitado. Un elemento se encuentra deshabilitado si no puede ser activado (por ejemplo ser selecionado, hacer click en él o aceptar texto de entrada) ni aceptar el foco. El elemento tiene además un estado habilitado en el cual puede ser activado o recibir foco.

:invalid

se utiliza para aplicar estilos a los campos de formulario que no cumplen con los requisitos de validación establecidos , como campos de texto con valores no permitidos o campos obligatorios que están vacíos.

:buffering

se utiliza para aplicar estilos a elementos en función de la dirección del texto, ya sea de izquierda a derecha (LTR) o de derecha a izquierda (RTL). Esto es especialmente útil para diseñar sitios web multilingües.
